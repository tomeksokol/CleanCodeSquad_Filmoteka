# Project: Filmoteka 🎥

Our team project is a website that searches for **popular movies** and allows you to create your own **movie library!**

## Used technologies and solutions: 💻

 - HTML , 
 - CSS, 
 - SASS, 
 - JavaScript, 
 - Snowpack, 
 - Public API, 
 - localStorage,
 - pagination, 
 - modal window, 
 - loader.
